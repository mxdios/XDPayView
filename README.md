# XDPayView

代码实现金额输入框，添加约束金额的输入限制。

- 只能输入一位小数点
- 第一位是0，第二位不是小数点自动覆盖0
- 小数点后最多只能输入两位数字
- 第一位输入小数点，前面自动补0

代码实现密码输入框

- 密码输入完成检测密码正确性
- 密码只能输入6位数字
- 密码输入错误可重新点击密码输入框输入
- 不允许输入框密码拷贝
- 密码密文处理
